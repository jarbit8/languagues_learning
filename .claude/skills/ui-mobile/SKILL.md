NAVEGACIÓN: bottom tab bar de 5: Inicio · Aprender · Examen · Hablar · Progreso. "Aprender" agrupa Vocabulario y Gramática del tema en curso. Listening dentro de "Hablar" o como sección propia.
PANTALLAS: 1) Inicio: tema en curso con % (vocab + gramática EN/FR), contador "hoy marcaste N palabras", botón Examen diario con badge de repasos SRS vencidos, siguiente paso lógico. 2) Vocabulario: tarjetas del tema con estado, tarjeta expandida = es+en+fr+audios+ejemplos+botón Aprendida. 3) Gramática: las dos lecciones (EN/FR) con regla, ejemplos con audio, 15 ejercicios. 4) Examen: los 4 tipos con estado (disponible/bloqueado y qué falta); una pregunta por pantalla. 5) Hablar: chat con IA, burbujas, 🔊 por mensaje, selector de escenario. 6) Progreso: mapa de 24 temas por bloque (verde/en curso/bloqueado), palabras dominadas/en repaso/débiles, historial de exámenes, racha (motivacional, nunca culpa). 7) Ajustes (dentro de Progreso): campo API key de Anthropic (input password → localStorage, aviso "queda solo en este dispositivo, nunca la subas al repo, es público"), verificación de voces TTS, velocidad de audio, tema claro/oscuro, exportar/importar progreso JSON desde Dexie.
PWA: vite-plugin-pwa, manifest nombre "Idiomas", display standalone, portrait, iconos 192/512, start_url y scope respetando base '/languagues_learning/'. Service worker registerType autoUpdate, precache del shell y data packs (offline total); llamadas a api.anthropic.com siempre por red, nunca cacheadas.
DISEÑO: mobile-first 390px, touch targets ≥44px, tipografía grande, modo oscuro según sistema, UI en español directo e informal, colores consistentes por idioma en toda la app.
