NAVEGACIÓN: bottom tab bar de 5: Inicio · Aprender · Examen · Practicar · Progreso (`components/TabBar.tsx`). "Aprender" agrupa Vocabulario y Gramática del tema en curso. "Practicar" (antes "Hablar", renombrado 2026-07-18, ruta sigue siendo `/hablar`, icono 🎯) es el hub de las 4 HABILIDADES en pestañas: 🗣️ Hablar (Conversación IA) · 🎧 Escuchar (Listening in-app) · 📖 Leer (Reading) · ✍️ Escribir (Writing). `screens/Hablar.tsx` es solo el contenedor de pestañas; cada habilidad es su propia pantalla (`Conversacion`, `Listening`, `Reading`, `Writing`).
PANTALLAS: 1) Inicio: tema en curso con % (vocab + gramática EN/FR), contador "hoy marcaste N palabras", botón Examen diario con badge de repasos SRS vencidos, siguiente paso lógico. 2) Vocabulario: tarjetas del tema con estado, tarjeta expandida = es+en+fr+audios+ejemplos+botón Aprendida. 3) Gramática: las dos lecciones (EN/FR) con regla, ejemplos con audio, 15 ejercicios. 4) Examen: los 4 tipos con estado (disponible/bloqueado y qué falta); una pregunta por pantalla. 5) Practicar (hub de 4 pestañas): Hablar = prompt copiable para IA externa (ver speaking-ai); Escuchar = diálogos in-app con audio + duración + preguntas (ver listening-engine); Leer = textos del bloque + preguntas de comprensión (ExamRunner); Escribir = consignas del bloque, escribir + corrección IA o autoevaluación con modelo (reusa `EscribirConsigna` de PasoWriting). Leer/Escribir eligen BLOQUE (1..bloqueEnCurso), no tema — su contenido es por bloque. 6) Progreso: mapa de 24 temas por bloque (verde/en curso/bloqueado), palabras dominadas/en repaso/débiles, historial de exámenes, racha (motivacional, nunca culpa). 7) Ajustes (dentro de Progreso): campo API key de Anthropic (input password → localStorage, aviso "queda solo en este dispositivo, nunca la subas al repo, es público"), verificación de voces TTS, velocidad de audio, tema claro/oscuro, exportar/importar progreso JSON desde Dexie.
PWA: vite-plugin-pwa, manifest nombre "Idiomas", display standalone, portrait, iconos 192/512, start_url y scope respetando base '/languagues_learning/'. Service worker registerType autoUpdate, precache del shell y data packs (offline total); llamadas a api.anthropic.com siempre por red, nunca cacheadas.
DISEÑO: mobile-first 390px, touch targets ≥44px, tipografía grande, modo oscuro según sistema, UI en español directo e informal, colores consistentes por idioma en toda la app.
