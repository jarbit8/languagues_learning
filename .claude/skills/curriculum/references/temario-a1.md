# Temario A1 — Inglés y Francés en paralelo

24 temas · 4 bloques de 6 · ~30 conceptos de vocabulario por tema. Cada tema: función comunicativa, vocabulario dual (es/en/fr), gramática EN, gramática FR, pronunciación, escenario de speaking. El orden codifica dependencias: ejercicios y diálogos solo usan vocabulario de este tema y anteriores.

---

## BLOQUE 1 — Yo y mi gente

### TEMA 1 — Saludos y presentaciones
**Función:** saludar, despedirte, presentarte, ser cortés.
**Vocabulario (30):** hello / hi / good morning / good afternoon / good evening / good night / goodbye / bye / see you later / see you soon / see you tomorrow / please / thank you / you're welcome / sorry / excuse me / yes / no / Mr / Mrs / name / nice to meet you / how are you / fine / welcome — con sus pares FR: bonjour / salut / bonsoir / bonne nuit / au revoir / à plus tard / à bientôt / à demain / s'il vous plaît / merci / de rien / pardon / excusez-moi / oui / non / monsieur / madame / nom / enchanté / comment ça va / bien / bienvenue.
**Gramática EN:** pronombres sujeto (I, you, he, she, it, we, they) + verbo *to be* afirmativo con contracciones (I'm, you're, he's, she's, it's, we're, they're). Uso de *my name is / I am*.
**Gramática FR:** pronoms sujets (je, tu, il, elle, on, nous, vous, ils, elles) + *être* completo (je suis, tu es, il/elle est, nous sommes, vous êtes, ils/elles sont). Diferencia tu/vous — el TEF Canada usa siempre *vous* en registro formal. *Je m'appelle*.
**Pronunciación:** EN — la *h* aspirada de *hello, how* (no muda como en español). FR — la *e* final muda (*madame* suena "madam"), la nasal de *bonjour*.
**Speaking:** presentarte con nombre y cortesía básica ("Hello, my name is… Nice to meet you"). Escenario: primer día conociendo a un compañero.

### TEMA 2 — Números 0-100 y edad
**Función:** contar, decir tu edad, dar tu número de teléfono.
**Vocabulario (30):** number / zero / one … ten / eleven / twelve / thirteen / twenty / thirty / forty / fifty / hundred / age / how old / year / years old / phone number / birthday / to have — con pares FR: numéro / zéro / un … dix / onze / douze / treize / vingt / trente / quarante / cinquante / cent / âge / quel âge / an / ans / numéro de téléphone / anniversaire / avoir.
**Gramática EN:** *to be* negativo (I'm not, isn't, aren't) e interrogativo (Are you…? Is he…?). *How old are you? I'm 22.* (edad con *to be*).
**Gramática FR:** *avoir* completo (j'ai, tu as, il/elle a, nous avons, vous avez, ils/elles ont). Edad con *avoir* (*j'ai 22 ans* — nunca *je suis*). Negación *ne… pas* (je n'ai pas).
**Pronunciación:** EN — acento en la sílaba de decenas (THIR-teen vs THIR-ty). FR — nasales (un, cent) y liaison (deux_ans, vingt_et_un).
**Speaking:** intercambiar edad y teléfono. Escenario: llenar una ficha de inscripción.

### TEMA 3 — Países, nacionalidades e idiomas
**Función:** decir de dónde eres, tu nacionalidad y qué idiomas hablas.
**Vocabulario (30):** country / Peru / Canada / France / England / Spain / nationality / Peruvian / Canadian / language / English / French / Spanish / to live / to speak / to come from / city / where — con pares FR: pays / Pérou / Canada / France / Angleterre / Espagne / nationalité / péruvien / canadien / langue / anglais / français / espagnol / habiter / parler / venir de / ville / où.
**Gramática EN:** artículos *a / an / the* (a country, an English book, the city). *Where are you from? I'm from Peru.*
**Gramática FR:** género de países (le Pérou, la France, les États-Unis) + preposiciones *en / au / aux* + país (*en France, au Canada, aux États-Unis*) y *à* + ciudad (*à Arequipa*). *C'est* vs *il est* (c'est un pays / il est canadien).
**Pronunciación:** EN — terminación *-ian* (Peruvian, Canadian). FR — distinción anglais/anglaise (concordancia audible).
**Speaking:** decir origen, nacionalidad e idiomas. Escenario: conocer a alguien de otro país.

### TEMA 4 — Familia
**Función:** hablar de tu familia y estado civil.
**Vocabulario (30):** family / father / mother / parents / brother / sister / son / daughter / husband / wife / grandfather / grandmother / uncle / aunt / cousin / child / children / single / married — con pares FR: famille / père / mère / parents / frère / sœur / fils / fille / mari / femme / grand-père / grand-mère / oncle / tante / cousin / enfant / enfants / célibataire / marié.
**Gramática EN:** adjetivos posesivos (my, your, his, her, its, our, their) + genitivo *'s* (my brother's name).
**Gramática FR:** possessifs (mon, ma, mes, ton, ta, tes, son, sa, ses, notre, votre, leur) + la posesión con *de* en vez de *'s* (le frère de Marie). *Mon / ton / son* ante palabra femenina que empieza por vocal (mon amie).
**Pronunciación:** EN — *th* de *father, mother, brother*. FR — la diferencia père/mère (è abierta).
**Speaking:** describir a tu familia. Escenario: mostrar una foto familiar a un amigo.

### TEMA 5 — Descripción física y personalidad
**Función:** describir cómo es alguien por fuera y por dentro.
**Vocabulario (30):** tall / short / young / old / beautiful / handsome / thin / fat / hair / eyes / kind / nice / funny / shy / smart / happy / sad / tired / blond / brown — con pares FR: grand / petit / jeune / vieux / beau / belle / mince / gros / cheveux / yeux / gentil / sympa / drôle / timide / intelligent / heureux / triste / fatigué / blond / brun.
**Gramática EN:** adjetivos invariables antes del sustantivo (a tall man, tall women) + plurales irregulares (man/men, woman/women, child/children, person/people).
**Gramática FR:** concordancia del adjetivo (género y número: grand/grande/grands/grandes) y su posición (normalmente después del sustantivo; beau/joli/grand/petit antes). Plural *-eau → -eaux* (beau/beaux).
**Pronunciación:** EN — vocales largas/cortas (short vs shirt). FR — beau/belle, vieux/vieille (formas masculina/femenina muy distintas).
**Speaking:** describir a un familiar o amigo. Escenario: describir a alguien para que otro lo reconozca.

### TEMA 6 — Profesiones y ocupaciones
**Función:** decir en qué trabajas o estudias y preguntarlo.
**Vocabulario (30):** job / work / student / teacher / doctor / nurse / engineer / programmer / waiter / driver / police officer / cook / seller / office / company / to work / to study / what / who / where — con pares FR: travail / travailler / étudiant / professeur / médecin / infirmier / ingénieur / programmeur / serveur / chauffeur / policier / cuisinier / vendeur / bureau / entreprise / étudier / quoi / qui / où.
**Gramática EN:** question words (what, who, where, when, why, how) + profesión con artículo (*I'm a programmer*, siempre con *a/an*).
**Gramática FR:** *est-ce que* + qui / que / où / quand + profesión SIN artículo (*je suis programmeur*, no *un programmeur*) + femenino de profesiones (vendeur/vendeuse, infirmier/infirmière).
**Pronunciación:** EN — *-er* final átona (teacher, driver). FR — femeninos audibles (étudiant/étudiante).
**Speaking:** decir tu profesión/estudios y preguntar la del otro. Escenario: charla de networking.
**→ Cierre Bloque 1: examen de bloque (4 habilidades).**

## BLOQUE 2 — Mi día a día

### TEMA 7 — Días, meses, estaciones y fechas
**Función:** decir el día, la fecha, tu cumpleaños y las estaciones.
**Vocabulario (30):** Monday…Sunday / January…December / spring / summer / autumn / winter / day / week / month / year / today / tomorrow / yesterday / date / weekend — con pares FR: lundi…dimanche / janvier…décembre / printemps / été / automne / hiver / jour / semaine / mois / année / aujourd'hui / demain / hier / date / week-end.
**Gramática EN:** preposiciones de tiempo *in / on / at* (in July, on Monday, at the weekend) + ordinales para fechas (the third of May, May 3rd).
**Gramática FR:** la fecha con cardinal (*le 3 mai*), excepto *le premier* (le 1er mai) + *en* + mes (*en mai*) + *le* + día para rutina (le lundi = los lunes).
**Pronunciación:** EN — Wednesday ("wenz-day"). FR — meses con nasales (janvier, novembre).
**Speaking:** decir la fecha de hoy y tu cumpleaños. Escenario: acordar el día de una reunión.

### TEMA 8 — La hora y la frecuencia
**Función:** decir la hora y con qué frecuencia haces cosas.
**Vocabulario (30):** time / hour / minute / o'clock / half past / quarter past / quarter to / noon / midnight / morning / afternoon / evening / night / always / usually / often / sometimes / never / early / late — con pares FR: heure / minute / et demie / et quart / moins le quart / midi / minuit / matin / après-midi / soir / nuit / toujours / d'habitude / souvent / parfois / jamais / tôt / tard.
**Gramática EN:** decir la hora (half past seven, quarter to nine, it's ten o'clock) + posición de los adverbios de frecuencia (I always study / I am never late).
**Gramática FR:** la hora (*il est sept heures et demie, huit heures moins le quart*) + adverbes de fréquence (je mange toujours…, je ne suis jamais…).
**Pronunciación:** EN — reducción "quarter" ("kwor-ter"). FR — liaison en *deux heures*, *trois heures*.
**Speaking:** decir tu horario de un día. Escenario: preguntar a qué hora abre un lugar.
**IMPORTANTE:** desde este tema en adelante todo examen de listening incluye preguntas tipo "escucha el audio y anota la hora" — de lo más evaluado en IELTS/TEF básico.

### TEMA 9 — Rutina diaria
**Función:** contar lo que haces cada día.
**Vocabulario (30):** to get up / to wake up / to have breakfast / to have lunch / to have dinner / to go to work / to come back / to study / to sleep / to shower / to brush / to start / to finish / to rest / breakfast / lunch / dinner / homework — con pares FR: se lever / se réveiller / prendre le petit-déjeuner / déjeuner / dîner / aller au travail / rentrer / étudier / dormir / se doucher / se brosser / commencer / finir / se reposer / petit-déjeuner / déjeuner / dîner / devoirs.
**Gramática EN:** presente simple completo + la *-s* de tercera persona (he studies, she goes) + auxiliares *do/does* en negativo e interrogativo (I don't work, does he study?).
**Gramática FR:** verbos regulares *-er* completos (je parle, tu parles, il parle, nous parlons, vous parlez, ils parlent) + verbos pronominales (je me lève, tu te lèves, il se lève).
**Pronunciación:** EN — tercera persona -s/-es sonora (goes, watches). FR — terminaciones -e/-es/-ent mudas (parle=parles=parlent).
**Speaking:** describir tu rutina de estudiante. Escenario: contarle tu día a un amigo.

### TEMA 10 — Casa, habitaciones y muebles
**Función:** describir dónde vives y qué hay en tu casa.
**Vocabulario (30):** house / apartment / room / kitchen / bedroom / bathroom / living room / door / window / table / chair / bed / sofa / wall / floor / there is / there are / near / next to / under / on — con pares FR: maison / appartement / pièce / cuisine / chambre / salle de bains / salon / porte / fenêtre / table / chaise / lit / canapé / mur / sol / il y a / près de / à côté de / sous / sur.
**Gramática EN:** *there is / there are* (afirmativo, negativo, interrogativo) + preposiciones de lugar (in, on, under, next to, between).
**Gramática FR:** *il y a* (invariable) + prépositions de lieu (dans, sur, sous, à côté de, entre) + contracciones *au/aux/du/des* (à côté du lit, près des fenêtres).
**Pronunciación:** EN — "there's" reducido. FR — il y a ("ilya"), enlace fluido.
**Speaking:** describir tu cuarto. Escenario: describir tu casa a alguien que la visitará.

### TEMA 11 — Objetos cotidianos y colores
**Función:** nombrar objetos y sus colores, señalar cosas.
**Vocabulario (30):** thing / phone / computer / book / pen / pencil / bag / key / watch / glasses / money / wallet / white / black / red / blue / green / yellow / this / that — con pares FR: chose / téléphone / ordinateur / livre / stylo / crayon / sac / clé / montre / lunettes / argent / portefeuille / blanc / noir / rouge / bleu / vert / jaune / ce / cette.
**Gramática EN:** demostrativos *this / that / these / those* + color invariable antes del sustantivo (a red bag, red bags).
**Gramática FR:** démonstratifs *ce / cet / cette / ces* (ce livre, cet ordinateur, cette clé, ces stylos) + colores con concordancia (un sac vert, une porte verte).
**Pronunciación:** EN — this/these (vocal corta/larga). FR — cet ante vocal (liaison).
**Speaking:** describir los objetos de tu mochila. Escenario: reportar un objeto perdido.

### TEMA 12 — Ropa
**Función:** describir qué lleva puesto alguien y qué haces ahora.
**Vocabulario (30):** clothes / shirt / t-shirt / trousers / jeans / dress / skirt / jacket / coat / shoes / socks / hat / sweater / to wear / to put on / new / old / to buy / size — con pares FR: vêtements / chemise / t-shirt / pantalon / jean / robe / jupe / veste / manteau / chaussures / chaussettes / chapeau / pull / porter / mettre / nouveau / vieux / acheter / taille.
**Gramática EN:** presente continuo (I'm wearing, she's wearing) + *have got* para posesión (I've got a jacket).
**Gramática FR:** verbo *porter* + interrogativos *quel / quelle / quels / quelles* (quelle robe? quels vêtements?).
**Pronunciación:** EN — -ing final (wearing). FR — quel/quelle mismo sonido, concordancia escrita.
**Speaking:** describir tu ropa de hoy. Escenario: comprar ropa y describir lo que buscas.
**→ Cierre Bloque 2: examen de bloque (con "anota la hora").**

## BLOQUE 3 — Comida y ciudad

### TEMA 13 — Comida básica, frutas y verduras
**Función:** hablar de comida y de lo que hay para comer.
**Vocabulario (30):** food / bread / rice / meat / chicken / fish / egg / cheese / apple / banana / orange / tomato / potato / onion / salad / fruit / vegetable / some / any / to eat — con pares FR: nourriture / pain / riz / viande / poulet / poisson / œuf / fromage / pomme / banane / orange / tomate / pomme de terre / oignon / salade / fruit / légume / du / de la / manger.
**Gramática EN:** sustantivos contables/incontables + *some / any* (some bread, any apples, is there any milk?).
**Gramática FR:** partitivos *du / de la / de l' / des* (du pain, de la viande, de l'eau, des fruits) + la regla de *de* tras negación (je ne mange pas de viande).
**Pronunciación:** EN — plural -s (apples, oranges). FR — œuf/œufs (f muda en plural).
**Speaking:** decir qué comes normalmente. Escenario: decir qué hay en tu cocina.

### TEMA 14 — Bebidas y restaurante
**Función:** pedir de comer y beber en un restaurante.
**Vocabulario (30):** water / coffee / tea / milk / juice / wine / beer / menu / bill / waiter / table / plate / glass / hungry / thirsty / to drink / to order / to want / breakfast / dessert — con pares FR: eau / café / thé / lait / jus / vin / bière / menu / addition / serveur / table / assiette / verre / faim / soif / boire / commander / vouloir / petit-déjeuner / dessert.
**Gramática EN:** *would like* para pedir cortésmente (I'd like a coffee, would you like…?).
**Gramática FR:** *je voudrais* + verbo *prendre* (je prends, vous prenez) + *avoir faim / avoir soif* (j'ai faim, tu as soif).
**Pronunciación:** EN — "I'd like" reducido. FR — voudrais (r gutural), liaison vous_avez.
**Speaking:** pedir una comida completa. Escenario: mesero y cliente en un café.

### TEMA 15 — Compras, dinero y precios
**Función:** comprar cosas, preguntar precios y cantidades.
**Vocabulario (30):** shop / store / market / price / money / cash / card / cheap / expensive / to pay / to cost / to sell / how much / how many / a lot / a little / change / receipt / customer / to need — con pares FR: magasin / boutique / marché / prix / argent / espèces / carte / bon marché / cher / payer / coûter / vendre / combien / beaucoup / un peu / monnaie / reçu / client / avoir besoin de.
**Gramática EN:** *how much* (incontables/precio) vs *how many* (contables) + *can* para pedir (Can I have…? Can I pay by card?).
**Gramática FR:** *combien* (combien ça coûte?) + expresiones de cantidad (beaucoup de, un peu de, trop de + sustantivo sin artículo).
**Pronunciación:** EN — "how much" / "how many". FR — combien (nasal final).
**Speaking:** comprar algo y preguntar el precio. Escenario: regatear en un mercado.

### TEMA 16 — Lugares de la ciudad
**Función:** nombrar lugares de la ciudad y decir qué se puede hacer.
**Vocabulario (30):** city / street / square / park / bank / hospital / school / university / library / museum / cinema / supermarket / pharmacy / church / station / hotel / restaurant / there / here / to visit — con pares FR: ville / rue / place / parc / banque / hôpital / école / université / bibliothèque / musée / cinéma / supermarché / pharmacie / église / gare / hôtel / restaurant / là / ici / visiter.
**Gramática EN:** *can / can't* completo para posibilidad y permiso (You can park here, you can't smoke).
**Gramática FR:** *pouvoir* completo (je peux, tu peux, il peut, nous pouvons, vous pouvez, ils peuvent) + *on* impersonal (on peut visiter le musée).
**Pronunciación:** EN — can/can't (la t muda pero cambia la vocal). FR — peux/peut mismo sonido.
**Speaking:** decir qué hay en tu ciudad y qué se puede hacer. Escenario: recomendar lugares a un turista.

### TEMA 17 — Direcciones y transporte
**Función:** pedir y dar direcciones, hablar de transporte.
**Vocabulario (30):** left / right / straight / corner / near / far / to turn / to go / to take / bus / train / car / bike / taxi / plane / metro / on foot / stop / ticket / map — con pares FR: gauche / droite / tout droit / coin / près / loin / tourner / aller / prendre / bus / train / voiture / vélo / taxi / avion / métro / à pied / arrêt / billet / plan.
**Gramática EN:** imperativo (Turn left, go straight, take the bus) + medios de transporte (by bus, by car, on foot).
**Gramática FR:** impératif (tourne/tournez, va/allez) + *aller* completo (je vais, tu vas, il va, nous allons, vous allez, ils vont) + *en bus / à pied / à vélo*.
**Pronunciación:** EN — "straight" (grupo str). FR — aller/vais/vont formas muy distintas.
**Speaking:** dar indicaciones para llegar a un lugar. Escenario: explicar cómo llegar a tu casa.

### TEMA 18 — Clima y naturaleza
**Función:** hablar del clima y del entorno natural.
**Vocabulario (30):** weather / sun / rain / wind / snow / cloud / hot / cold / warm / cool / sunny / rainy / windy / cloudy / sea / mountain / river / tree / sky / beach — con pares FR: temps / soleil / pluie / vent / neige / nuage / chaud / froid / doux / frais / ensoleillé / pluvieux / venteux / nuageux / mer / montagne / rivière / arbre / ciel / plage.
**Gramática EN:** *What's the weather like?* + *It's sunny / it's raining* (adjetivo vs continuo).
**Gramática FR:** *Quel temps fait-il?* + verbo *faire* completo (je fais, tu fais, il fait, nous faisons, vous faites, ils font) usado para clima (il fait chaud, il pleut).
**Pronunciación:** EN — weather/whether. FR — faisons ("fezon"), excepción de pronunciación.
**Speaking:** describir el clima de tu ciudad por estación. Escenario: comparar el clima de Arequipa y Canadá.
**→ Cierre Bloque 3: examen de bloque.**

## BLOQUE 4 — Pasado, planes y gustos

### TEMA 19 — Cuerpo y salud básica
**Función:** nombrar partes del cuerpo y decir qué te duele.
**Vocabulario (30):** body / head / eye / ear / nose / mouth / tooth / neck / arm / hand / leg / foot / stomach / back / heart / sick / pain / doctor / medicine / to hurt — con pares FR: corps / tête / œil / oreille / nez / bouche / dent / cou / bras / main / jambe / pied / ventre / dos / cœur / malade / douleur / médecin / médicament / avoir mal.
**Gramática EN:** *have got* para síntomas (I've got a headache, she's got a cold).
**Gramática FR:** *avoir mal à* + contracciones (j'ai mal à la tête, au ventre, aux dents) + *devoir* (je dois me reposer).
**Pronunciación:** EN — tooth/teeth, foot/feet (plurales irregulares). FR — mal à la / au / aux (contracción audible).
**Speaking:** decir qué te duele. Escenario: explicar un síntoma en la farmacia.

### TEMA 20 — Hobbies, deportes y música
**Función:** hablar de lo que te gusta hacer en tu tiempo libre.
**Vocabulario (30):** hobby / sport / football / basketball / swimming / running / music / guitar / to play / to sing / to dance / to read / to watch / to listen / to like / to love / to hate / free time / gym / film — con pares FR: loisir / sport / football / basket / natation / course / musique / guitare / jouer / chanter / danser / lire / regarder / écouter / aimer / adorer / détester / temps libre / gym / film.
**Gramática EN:** *like / love / hate + verb-ing* (I like reading) + *play / go / do* con deportes (play football, go swimming, do yoga).
**Gramática FR:** *aimer / adorer / détester + infinitivo* (j'aime lire) + *faire du / de la* (je fais du sport, de la natation) + *jouer à / de* (jouer au foot, jouer de la guitare).
**Pronunciación:** EN — -ing en gerundios. FR — jouer à/de (distinción deporte/instrumento).
**Speaking:** decir tus hobbies. Escenario: proponer una actividad a un amigo.

### TEMA 21 — Universidad y estudios
**Función:** hablar de tu carrera, materias y por qué estudias.
**Vocabulario (30):** university / class / subject / exam / homework / grade / classroom / computer science / math / to learn / to teach / to pass / to fail / difficult / easy / because / so / to understand / library / degree — con pares FR: université / cours / matière / examen / devoirs / note / salle de classe / informatique / maths / apprendre / enseigner / réussir / échouer / difficile / facile / parce que / donc / comprendre / bibliothèque / diplôme.
**Gramática EN:** object pronouns (me, you, him, her, it, us, them) + conectores *because / so* (I study because I like it, so I practice).
**Gramática FR:** pronoms toniques (moi, toi, lui, elle, nous, vous, eux, elles) + *parce que / donc* (j'étudie parce que j'aime ça, donc je pratique).
**Pronunciación:** EN — them/they. FR — eux ("ø"), lui.
**Speaking:** hablar de tu carrera (Ciencia de la Computación) y por qué. Escenario: entrevista de admisión.

### TEMA 22 — Tecnología y comunicación
**Función:** hablar de tecnología y contar algo que pasó (pasado inicial).
**Vocabulario (30):** internet / email / message / app / website / screen / password / to download / to send / to call / to text / to click / to open / online / network / file / photo / video / yesterday / last — con pares FR: internet / e-mail / message / appli / site web / écran / mot de passe / télécharger / envoyer / appeler / envoyer un texto / cliquer / ouvrir / en ligne / réseau / fichier / photo / vidéo / hier / dernier.
**Gramática EN:** pasado de *to be* (was/were) + past simple regular con *-ed* (I worked, she downloaded, they played).
**Gramática FR:** introducción al *passé composé* con *avoir* + participio regular *-é* (j'ai parlé, tu as téléchargé, il a envoyé).
**Pronunciación:** EN — las 3 pronunciaciones de -ed (/t/ /d/ /ɪd/: worked, played, wanted). FR — participio -é (parlé="parlé").
**Speaking:** contar qué hiciste ayer con tu celular/PC. Escenario: contarle a alguien un problema técnico.

### TEMA 23 — Viajes, aeropuerto y hotel
**Función:** hablar de viajes y contar experiencias pasadas.
**Vocabulario (30):** trip / travel / airport / flight / plane / passport / luggage / suitcase / hotel / room / booking / ticket / to arrive / to leave / to stay / to visit / abroad / holiday / border / to pack — con pares FR: voyage / voyager / aéroport / vol / avion / passeport / bagages / valise / hôtel / chambre / réservation / billet / arriver / partir / rester / visiter / à l'étranger / vacances / frontière / faire les valises.
**Gramática EN:** top 15 pasados irregulares (went, had, did, saw, came, got, took, made, said, ate, bought, found, gave, knew, was/were).
**Gramática FR:** *passé composé* con *être* para verbos de movimiento (je suis allé, il est parti, nous sommes arrivés — con concordancia) + *venir de + infinitivo* (je viens d'arriver).
**Pronunciación:** EN — pasados irregulares (went, bought "bawt"). FR — concordancia audible (allée, allés).
**Speaking:** contar un viaje que hiciste. Escenario: check-in en el hotel / control migratorio (útil para Canadá).

### TEMA 24 — Planes, gustos y conectores (cierre)
**Función:** hablar de tus planes futuros e integrar todo lo aprendido.
**Vocabulario (30):** plan / future / to want / to hope / to plan / to go to / next / soon / project / dream / goal / to move / to travel / to study / and / but / or / because / then / also — con pares FR: projet / futur / vouloir / espérer / planifier / aller / prochain / bientôt / projet / rêve / but / déménager / voyager / étudier / et / mais / ou / parce que / puis / aussi.
**Gramática EN:** *going to* para planes (I'm going to move to Canada) + repaso de question words.
**Gramática FR:** *futur proche* (aller + infinitivo: je vais déménager) + repaso de mots interrogatifs.
**Pronunciación:** EN — "gonna" (going to reducido, reconocerlo). FR — je vais + infinitivo (fluidez).
**Speaking:** contar tu plan de emigrar a Canadá. Escenario: entrevista sobre tus metas (formato IELTS Part 2/TEF EO).
**→ Cierre Bloque 4: examen de bloque → examen final A1 (85% vocab / 80% habilidades).**
